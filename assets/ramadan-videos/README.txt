VIDEOS RAMADAN - BARAKAAH APP
==============================

Placez vos videos dans les dossiers suivants :

/dua/
  - Videos de Dua (invocations)
  - Format : .mp4
  - Exemple : dua-iftar.mp4, dua-suhoor.mp4

/ramadan/
  - Videos generales sur le Ramadan
  - Format : .mp4
  - Exemple : conseils-ramadan.mp4, recitation.mp4

IMPORTANT :
- Seuls les fichiers .mp4 sont supportes
- Le nom du fichier sera utilise comme titre
- Utilisez des tirets (-) au lieu des espaces
