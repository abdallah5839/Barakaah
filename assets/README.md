# Assets

Ce dossier contient les ressources visuelles de l'application.

## Fichiers requis

Créez les fichiers suivants :

### icon.png
- Taille : 1024x1024 pixels
- Format : PNG
- Utilisé comme icône de l'application

### splash.png
- Taille : 1284x2778 pixels (recommandé)
- Format : PNG
- Écran de chargement de l'application

### adaptive-icon.png
- Taille : 1024x1024 pixels
- Format : PNG
- Icône adaptative pour Android

### favicon.png
- Taille : 48x48 pixels
- Format : PNG
- Icône pour la version web

## Création rapide

Pour un développement rapide, vous pouvez créer des images placeholder avec un fond vert (#059669) et le texte "Barakaah".

Utilisez un outil comme :
- Canva
- Figma
- Adobe XD
- Ou tout éditeur d'images
